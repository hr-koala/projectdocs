# Finance 相关

### 八类不碰

- 短期大涨的，不碰
- 高位除权的，不碰
- 换手率低于 1%的，不碰
- 任何有问题的，不碰
- 技术指标全部走坏的，不碰
- 看一只票最近 3 个月内的走势
- 高位出利好，高位复牌，高位加速的，不碰
- 暴涨暴跌的，不碰

### 六不抛

- 处于上升趋势的，不要抛
- 处于近期热点的，不要抛
- 经过连续的无量下跌，突然放量大涨，且换手率大于 3%的，不要抛
- 长期在底部横盘震荡，突破后，又继续回到平台，继续震荡的，不要抛
- 基本面业绩优的，不要抛
- 技术面多个指标形成金叉共振的，不要抛

**_去理解主力思维，站在主力的角度去逆向思考，领悟了人性，就能抓住股性_**。
